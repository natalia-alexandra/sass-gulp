## Gulp (Task runner)

### Installation & Setup

1. Install gulp globally

```
npm install -g gulp
```

2. Install gulp locally in the project

```
npm install gulp
```

3. Add below line in your `package.json`

```
"scripts": { "gulp": "gulp" }
```

4. Run gulp

```js
npm run gulp
```
